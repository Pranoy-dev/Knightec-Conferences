Debugging Steps:

  1. Open browser DevTools (F12 or Cmd+Option+I)
  2. Go to Elements/Inspector tab
  3. Search for: data-sort-by-date-button
  4. If found: Button exists in DOM, check CSS/styling
  5. If NOT found: Button not rendering, check React/Next.js
  6. Check Console tab for any JavaScript errors
  7. Clear browser cache: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
  8. Clear Next.js cache: rm -rf .next && npm run dev
